# Ronin Chronicles
## Описание

Игра от третьего лица в жанре Action с элементами Hack and Slash.  



## Задачи:

* Найти 3D модели персонажей и окружения
* Создать анимации
* Создать боевую систему игрока
* Создать уровень (окружение)
* Найти и интегрировать в игру библиотеку звуков
* Написать ИИ игровых врагов
* Протестировать игру



## Управление

* W, A, S, D - перемещение
* Shift - Бег
* Q - Поглощение здоровья: поглощает здоровье ближайшего противика в поле зрения, восполняя его у игрока. Для использования кнопку необходимо зажимать, способность заканчивается либо по истемению её времени действия в 3 секунды либо по отжатию Q.
* ЛКМ - лёгкая атака (После 4-ой атаки в серии на игрока накладывается cooldown)
